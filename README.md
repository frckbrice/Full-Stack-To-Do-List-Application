# A Full-stack TODO List app

This is a FullStack to-do list application built with 

- **Client**: Angular 2+ in (`/client`)
- **Api**: NestJS in (`/api`)

## Getting Started 

### Front-end (Angular)

```bash
cd client
npm install
ng serve
```

### Back-end (Node API)

```bash
cd api
yarn install
yarn run start:dev
```

## Features
- Add, edit, Read and delete to-do items
- mark task as completed
- persistent storage (postgreSQL)
